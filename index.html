apire un pacchetto di figurine 1. definire pacchetto di fugurine -> (contenitore di plastica, quadrato, colorato, con stampa sopra ) - contiene figurine -> (foglietti di carta con stampa impressa sopra) 2. apri il pacchetto tirando il lembo in alto verso destra 3. sfila le carte dal pacchetto e definisci quante carte hai nel mazzo 4. conta quante carte hai e dividi le carte tra quelle nuove e i doppioni IF -> carta nuova -> metti nella lista 'nuove' ELSE -> metti nella lista 'doppioni' WHILE -> ci sono carte nel mazzo ripeti il ciclo dal punto 4. 5. Definisci 'Album' -> (Contenitore rigido di carta che contiene la collezione di carte possedute) 5. inserisci le carte della lista 'nuove' nell'album 6. conta quante carte ci sono nella lista 'doppioni' - raggruppa le carte della lista 'doppioni' in gruppi di carte uguali - conta quante carte ci sono in ciascun gruppo IF -> ci sono abbastanza carte in ciascun gruppo (superiore a 5) -> metti nella lista 'carte x scambio' ELSE -> metti nella lista -> 'carte non scambiabili' WHILE -> ci sono carte nel mazzo 'doppioni' continua il ciclo 6.
